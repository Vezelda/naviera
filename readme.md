# Logística Fluvial PY — Sitio Web

Proyecto de sitio web corporativo para empresa de abastecimiento fluvial y distribución frutihortícola en Paraguay.

## Integrantes

- Anibal Cardozo
- Oliver Demestri
- Bruno Sanchez
- Hugo Colman

## Estructura del proyecto

```
naviera/
├── index.html        # Página principal con hero y vista previa de secciones
├── servicios.html    # Detalle de servicios logísticos
├── noticias.html     # Artículos y novedades
├── acercade.html     # Información institucional
├── contacto.html     # Formulario y datos de contacto
├── assets/
│   └── css
│   │    └──    style.css     # Variables y estilos compartidos
│   └── img/        # Imágenes del sitio
```

## Tecnologías

- HTML5
- CSS3
- Google Fonts (Barlow Condensed + Lora)

## Cómo usar

1. Clonar el repositorio
2. Abrir `index.html` en el navegador
